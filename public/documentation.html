MyFlix Application

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Request Body Data Return Body Data
Movies List /movies GET none JSON object (data all movies)
Get data (1 movie) /movies/[title] GET none JSON object (data single movie [ID])
Get data (1 genre) /movies/genres/[Name] GET none Text description of genre [ID]
Get data 1 director /movies/directors/[Name] GET none JSON object (data 1 director [ID])
Add user /users POST JSON object (user info) JSON object(user info) + success message
Update User /users/[Username] PUT none Text message: successfully updated
Add movie to favorites /users/[Username]/favorites/MovieID PUT none Text message: List successfully updated
Remove movie from favorites /users/[Username]/favorites/MovieID DELETE none Text message: Movie successfully removed
De-register User /users/[Username] DELETE none Text message: User data successfully removed
